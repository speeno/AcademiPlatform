# 강사용 CMS 수정 체크리스트

## 기본 원칙

- 1차 등록본은 초안입니다. 강사가 문구/순서/예시를 실제 차수에 맞게 수정합니다.
- 공개 전에는 `DRAFT` 상태를 유지합니다.

## 레슨별 확인

- [ ] 레슨 제목이 실제 세션 제목과 일치
- [ ] HTML 본문 오탈자/링크 확인
- [ ] 라이브 세션 레슨(`LIVE_LINK`)에 실제 입장 URL 반영
- [ ] 워크북/템플릿 다운로드 링크 또는 제출 방법 명시
- [ ] 과제 설명/마감일(dueAt) 차수 일정에 맞게 조정

## 검수 요청 전

- [ ] Day별 학습 목표/산출물이 명확함
- [ ] 운영 안내(출석, 제출 방식, 문의 채널) 포함
- [ ] 민감 정보/내부 URL 노출 없음

## 검수/승인 후

- [ ] 강의실에서 표시 확인(`/classroom/{courseId}`)
- [ ] 샘플 계정으로 열람/과제 제출 테스트
- [ ] 최종 공지용 링크 정리
