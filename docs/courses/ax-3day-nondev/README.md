# [N3] AX 실무 3Day — 비개발자

> 3일 동안 입문부터 **팀 미니 프로젝트 완주**까지. 1일 과정의 맛보기를 넘어, 실제 동작하는 결과물을 팀으로 만들어 데모합니다.

## 한 줄 소개
> "3일이면, 팀이 함께 내 업무 문제를 푸는 도구를 만들어 시연한다."

---

## 운영 정보
| 항목 | 내용 |
|---|---|
| 대상 | 코딩 경험 적은 실무자(전 직군) |
| 기간 | 3일 (1일 6시간, 총 18시간) |
| 정원 | 12~20명 (4~5인 팀제) |
| 도구 | Cursor(메인) + Claude(채팅) |
| 형태 | 강의 30% + 실습·팀 프로젝트 70% |
| 수료 | 출석 + 일일 산출물 + **팀 데모** |

## 학습 성과
- AX·바이브 코딩·하네스·Agent Skills를 이해하고 사용한다.
- 반복 업무 절차를 `SKILL.md`·기술 파일로 만든다.
- 팀으로 **문제 정의 → 분해 → 빌드 → 데모**까지 완주한다(축약형).

## 3일 한눈에
| 일자 | 주제 | 끝나면 |
|---|---|---|
| Day 1 | 입문 & 첫 성공 | 동작하는 미니 도구 + 잘 시키는 법 |
| Day 2 | 하네스 & Agent Skills·기술 파일 | 다단계 빌드 + `SKILL.md` + 팀 구성 |
| Day 3 | 팀 미니 프로젝트(정의→빌드→데모) | **동작하는 팀 결과물 + 데모** |
