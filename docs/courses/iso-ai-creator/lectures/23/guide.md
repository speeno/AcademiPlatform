# 강 23 — 이미지 생성 실습 · 강의 스크립트

> AI Creator · ISO/IEC 17024 · 1강 패키지 (강사용 운영안) · 2차 실기 대비

## 강의 개요
- **시간**: 약 30분 (슬라이드 11장 + 실습)
- **목표**: Text-to-Image 4요소 + 반복 개선
- **선수**: 강16(NST 활용) / **산출물**: 이미지 프롬프트 + 개선 기록

## 시간 배분
| 구간 | 시간 | 슬라이드 |
|---|---|---|
| 도입·원리 | 7분 | 3~4 |
| 4요소·프롬프트 | 9분 | 5~7 |

| 정리·퀴즈 | 4분 | 10~11 |

## 도입 (멘트)
> "NST와 GAN이 발전해 이제 '글로 그림'을 그립니다. DALL·E, Midjourney가 대표죠. 핵심은 프롬프트 설계입니다."

## 전개 (슬라이드별 강의 포인트)
### S4 원리
- 확산 모델 개념, 프롬프트가 결과 좌우.
### S5~7 4요소·프롬프트
- 주제·스타일·구성·세부 + 네거티브 프롬프트.
### S8 실습
- 4요소 작성→3회 반복 개선.

## 실습 (8분)
- 주제 선정→4요소 프롬프트→반복 개선→최종본 제출.

## 정리 (멘트)
> "구체적 프롬프트 + 반복 개선이 핵심입니다. 다음 강은 스타일 전이·이미지 편집 실습입니다."

## 예상문제 + 해설
1. **(주관식)** 이미지 프롬프트 4요소? → 주제·스타일·구성·세부.
2. **(실기)** 주제로 4요소 프롬프트 작성 → 4요소 포함·구체성 채점.
> 채점 루브릭: 4요소(8)·반복 개선(7)·윤리 표시(5)·완성도(5).
