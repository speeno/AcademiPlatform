# 강 11
## GANs의 응용
AI Creator · ISO/IEC 17024 — 필기 대비

> "현실에 없는 것을 만드는 힘."

---

## 학습 목표
1. GANs의 **콘텐츠 산업** 응용을 안다
2. **합성 데이터** 의 가치를 이해한다
3. 과학·산업 응용을 설명한다

---

## 도입 — 양날의 검
- GANs는 현실에 없는 것을 창조
- 혁신적 응용 + 윤리적 그림자(다음 강)
- 오늘은 '놀라운 활용'에 집중

---

## ① 가상 세계·콘텐츠 산업
- 현실과 구분 어려운 **가상 인물·인플루언서**
- 게임 캐릭터·배경·아이템 자동 생성
- 마케팅·엔터테인먼트·교육 활용

---

## ② 합성 데이터 (Synthetic Data)
- 의료: 환자 **프라이버시 보호** 하며 영상 데이터 생성
- 자율주행: '아이가 뛰어드는' 희귀 위험 상황 생성
- 금융: 가상 거래 데이터로 사기 탐지 학습

---

## ③ 데이터 증강 (Augmentation)
- 수집이 어렵거나 비싼 데이터를 인공 생성
- 학습 데이터셋을 확장 → 성능 향상
- 희귀 케이스 보강에 유용

---

## ④ 과학·산업
- **신소재 개발**: 원하는 특성의 분자 구조 생성
- **신약 개발**: 후보 물질 탐색 자동화(기간 단축)
- 시뮬레이션용 가상 데이터 생성

---

## ⑤ 실용·예술
- 스타일 변환(사진→화풍), 흑백→컬러 복원
- 표정·배경 변경
- StyleGAN 기반 사실적 가상 인물

---

## 응용 한눈에
1. 가상 인물·콘텐츠
2. 합성 데이터
3. 데이터 증강
4. 신소재·신약
5. 예술·복원

---
