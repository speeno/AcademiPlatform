# 강 30 — AI Creator 2차 실기·주관식 모의고사 · 모범답안·채점기준

> AI Creator · ISO/IEC 17024 · 2차 실기 대비
> 형식: 주관식 + 실기(제작 계획) / 합격 **60점 이상**
> 과목 태그: [활용실제] [도구] [윤리] [이론]
> **이 문항은 플랫폼 문제은행(DB)에도 등록한다.**

---

## A. 주관식 (각 10점, 5문항 = 50점)

**1. [이론]** GANs의 생성자와 판별자의 역할을 각각 설명하고, 학습이 도달하는 최적 상태를 쓰시오.
**모범답안**: **생성자**는 무작위 잡음에서 진짜 같은 가짜를 만든다. **판별자**는 진짜/가짜를 구별한다. 둘의 경쟁(미니맥스)이 충분히 진행되면 판별자가 더 이상 구별하지 못하는 **내쉬 균형**에 도달한다.
**채점**: 생성자(3)+판별자(3)+내쉬 균형(4).

**2. [이론]** NST의 3대 손실 함수를 쓰고 각각의 역할을 설명하시오.
**모범답안**: **내용 손실**(형태/구조 유지, 깊은 층), **스타일 손실**(화풍 모방, 그람 행렬), **총 변동 손실**(인접 픽셀 매끄러움/정규화).
**채점**: 각 손실 명칭+역할(3·3·4).

**3. [이론]** AI 음악의 심볼릭 기반(MIDI)과 오디오 기반(Waveform)의 차이를 설명하시오.
**모범답안**: 심볼릭은 음의 높이·길이·세기 등 **기호(악보)** 를 다뤄 멜로디·화성 구조에 강하다. 오디오는 MP3/WAV 같은 **실제 소리 파형** 을 다뤄 음색·합성·음원 분리에 강하다.
**채점**: 심볼릭=기호(5)+오디오=파형(5).

**4. [활용]** GANs의 합성 데이터(Synthetic Data)가 가지는 가치를 의료 분야 예로 설명하시오.
**모범답안**: 실제 환자 데이터는 확보가 어렵고 프라이버시 문제가 있다. GANs로 실제와 구분 어려운 **합성 의료 영상**을 만들면, 개인정보를 노출하지 않고도 AI 진단 모델을 충분한 데이터로 학습시킬 수 있다.
**채점**: 프라이버시 보호(5)+데이터 부족 해결(5).

**5. [윤리]** AI 생성 콘텐츠에서 '표시 의무'와 '저작권 확인'이 필요한 이유를 서술하시오.
**모범답안**: **표시 의무**는 생성물을 진짜로 오인하거나 가짜 정보로 악용되는 것을 막아 신뢰를 지킨다. **저작권 확인**은 학습/사용된 원작자 권리 침해와 상업적 분쟁을 예방한다.
**채점**: 표시 의무(5)+저작권(5).

---

## B. 실기 — 제작 계획 작성 (각 25점, 2문항 = 50점)

**6. [활용실제/도구]** "친환경 브랜드 30초 홍보 영상"의 제작 계획을 작성하시오.
- 조건: 컷별 장면 프롬프트 + 사용할 AI 도구(이미지/영상/음악) + 편집·더빙 계획 + 윤리 점검 포함.

**모범답안(예)**:
```
[기획] 타깃: 2030, 메시지: 지속가능한 소비
[컷1] 드론 샷, 푸른 숲을 천천히 비행 (Text-to-Video)
[컷2] 텀블러 클로즈업, 따뜻한 색감 (Text-to-Image→영상)
[컷3] 아바타 내레이터가 슬로건 전달 (립싱크 아바타)
[음악] 잔잔한 어쿠스틱 BGM (AI 작곡, 로열티 프리)
[더빙] TTS 다국어 자막·음성
[윤리] 실존 인물 미사용·생성물 표시·음원 라이선스 확인
```
**채점 루브릭(25)**: 컷별 프롬프트(8)·도구 선택 적절성(6)·편집/더빙(6)·윤리(5).

**7. [활용/도구]** "가상 갤러리 메타버스 씬"의 제작 계획을 작성하시오.
- 조건: 3D 에셋 생성 방법 + 씬 구성(조명·동선·인터랙션) + 결합할 콘텐츠(이미지/음악) + 윤리·최적화 포함.

**모범답안(예)**:
```
[컨셉] 디지털 아트 가상 갤러리
[에셋] Text-to-3D로 전시 공간·좌대 생성, 작품은 Text-to-Image
[씬] Unity 배치, 따뜻한 스포트 조명, 관람 동선 설계, 작품 클릭 시 설명
[사운드] 차분한 앰비언트 BGM(AI 작곡)
[윤리·최적화] 저작물 무단 사용 금지·폴리곤 최적화·멀미 저감
```
**채점 루브릭(25)**: 에셋 생성(7)·씬 구성(8)·콘텐츠 결합(5)·윤리/최적화(5).

- 주관식 50 + 실기 50 = **100점**, **60점 이상 합격**.
- 실기는 루브릭 항목별 부분점수.
- 과목 태그별 득점 분포로 약점 진단.
