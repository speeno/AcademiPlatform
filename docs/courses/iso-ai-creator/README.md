# AI Creator — ISO/IEC 17024 국제자격 과정

> IQCS·AI GTC ISO/IEC 17024(AI 부문) **AI Creator** 자격 대비 교육 과정.
> 교재 『AI 전문가 국제자격증 과정 — AI Creator(크리에이터 과정 1)』(맨도롱북스, AI GTC) 기반.

## 과정 목표
- 1차 **필기** 합격을 위한 생성형 콘텐츠 AI 이론(딥러닝·GANs·NST·음악·VR/3D)을 체계적으로 익힌다(기본).
- 2차 **실기/제작** 합격을 위한 **이미지·영상·음악·3D 콘텐츠 제작** 역량을 기른다(심화).
- 자격 취득을 넘어 AI로 실제 콘텐츠를 창작하는 능력을 갖춘다.

## 과정 구성 (30강)
- **기본(필기 대비) 1~22강**: 교재 PART(딥러닝·GANs·NST·AI음악·AI애니메이션/VR)에 정렬한 이론 + 핵심개념·용어·예상문제.
- **심화(실기 대비) 23~30강**: 이미지·영상·음악·3D 제작 실습 + 저작권/윤리 + 캡스톤 + 2차 실기 모의.
