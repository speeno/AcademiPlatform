# AX 교육 과정군 (Course Family) — 마스터 인덱스 & 제작 계획

> 하네스 & Agent Skills 기반 바이브 코딩 교육을 **기간(1일/3일/7일) × 대상(비개발/개발)** 조합으로 제공하기 위한 전체 설계도입니다.
> 이미 완성된 **1주 비개발자 과정**을 기준 콘텐츠로 삼아, 나머지를 재단·확장합니다.

---

## 1. 과정 매트릭스
| 코드 | 과정 | 대상 | 기간 | 핵심 목표 | 상태 |
|---|---|---|---|---|---|
| N1 | AX 입문 1Day | 비개발 | 1일(6h) | 두려움 제거 + 첫 바이브 코딩 성공 + 스킬 맛보기 | ✅ 완료 (`ax-1day-nondev/`) |
| N3 | AX 실무 3Day | 비개발 | 3일(18h) | 입문 + 팀 미니 프로젝트(축약) | ✅ 완료 (`ax-3day-nondev/`) |
| **N7** | **AX 실무 1주** | **비개발** | **5일(30h)** | **문제정의→빌드→데모 풀코스** | **✅ 완료** |
| D1 | 에이전트 코딩 1Day | 개발 | 1일(6h) | 하네스·스킬로 개발 생산성 즉시 향상 | ✅ 완료 (`agent-coding-1day-dev/`) |
| D3 | 에이전트 코딩 3Day | 개발 | 3일(18h) | 하네스 심화 + 실전 워크플로우·자동화 | ✅ 완료 (`agent-coding-3day-dev/`) |
| D7 | 에이전트 코딩 1주 | 개발 | 5일(30h) | 커스텀 스킬·서브에이전트·MCP·자동화 풀코스 | ✅ 완료 (`agent-coding-1week-dev/`) |

> "7일"은 집중 1주(주 5일) 기준으로 설계하며, 7일(주말 포함) 일정이 필요하면 +2일 **확장 모듈**(심화 프로젝트/배포·확산)을 덧붙입니다.

## 2. 비개발 vs 개발 — 무엇이 다른가
| 축 | 비개발 트랙(N) | 개발 트랙(D) |
|---|---|---|
| 도구 | Cursor(메인) + Claude(채팅) | Cursor + **Claude Code(CLI)** + Claude API/Agent SDK |
| 깊이 | "잘 시키고 검증" 중심 | 하네스 내부 원리·컨텍스트 엔지니어링·자동화 |
| 스킬 | 글로 쓰는 `SKILL.md` | 스킬 라이브러리화·버전관리·팀 배포 |
| 확장 주제 | 안전·검증·업무 적용 | 서브에이전트, MCP 연동, 코드리뷰/테스트 자동화, CI 훅 |
| 결과물 | 동작하는 업무 도구 | 실제 레포에 적용한 워크플로우/자동화 |

## 3. 공통 자산(재사용) — `_shared/`
과정 간 중복 제작을 피하기 위해 공통 모듈을 한 곳에 두고 각 과정에서 참조합니다.
- `glossary.md` — 표준 용어(하네스·Agent Skills·바이브 코딩 등)
- `safety-ethics.md` — 개인정보·검증·안전 가이드
- `prompt-cookbook-nondev.md` / `prompt-cookbook-dev.md` — 대상별 프롬프트 치트시트
- `industry-examples.md` — 직군/영역별 프로젝트 소재·예시 프롬프트(비개발 9직군 + 개발 5영역)
- `templates/` — 문제정의 캔버스·분해표·SKILL 템플릿·팀차터·데모대본(공용)
> 현재 1주 비개발 과정의 `templates/`, `30-prompt-cookbook.md` 를 `_shared/`로 승격해 재사용.

```
docs/courses/
├─ README.md                     ← 본 문서(과정군 인덱스)
├─ _shared/                      ← 공통 자산
├─ harness-agent-skills-1week/   ← N7 (완료)
├─ ax-1day-nondev/               ← N1
├─ ax-3day-nondev/               ← N3

├─ agent-coding-3day-dev/        ← D3

```
각 과정 폴더는 N7과 동일한 골격(소개서·셋업·일자별 운영안·슬라이드·랩·평가·체크리스트)을 갖되 분량은 기간에 맞게 조절.

## 5. 과정별 커리큘럼 개요(초안)

### N1 — AX 입문 1Day (비개발)
오전: AX·바이브코딩 개요 → 잘 시키는 법 → 랩(미니 도구) / 오후: 하네스 맛보기 → `SKILL.md` 맛보기 → 내 업무 적용 아이디어 + 다음 단계.
→ N7의 Day1 + Day2·3 핵심 압축.

### N3 — AX 실무 3Day (비개발)
Day1 입문·첫 성공 / Day2 하네스 + 스킬·기술파일 / Day3 팀 미니 프로젝트(정의→빌드→데모).
→ N7을 3일로 압축(팀 프로젝트 범위 축소).

### D1 — 에이전트 코딩 1Day (개발)
에이전트 코딩 멘탈모델(하네스·도구·컨텍스트·루프) → 효과적 프롬프팅/플랜 모드 → 커스텀 Agent Skills 작성 → 실전 워크플로우(버그수정·리팩터·테스트) → 컨텍스트/서브에이전트 기초 → 안전·검증.

### D3 — 에이전트 코딩 3Day (개발)
Day1 멘탈모델+워크플로우 / Day2 스킬 라이브러리·서브에이전트·MCP 연동 / Day3 자동화 파이프라인(코드리뷰·테스트·CI 훅) + 팀 적용 프로젝트.

### D7 — 에이전트 코딩 1주 (개발)
D3 전체 + 컨텍스트 엔지니어링 심화, 멀티에이전트 오케스트레이션, 평가/관측, 사내 표준화·배포까지 풀코스 + 실전 프로젝트.

## 6. 제작 로드맵 — ✅ 전체 완료
권장 순서대로 모두 제작 완료했습니다:
1. ✅ N1 (1일 비개발) — `ax-1day-nondev/`
2. ✅ D1 (1일 개발) — `agent-coding-1day-dev/`
3. ✅ N3 (3일 비개발) — `ax-3day-nondev/` · ✅ D3 (3일 개발) — `agent-coding-3day-dev/`
4. ✅ D7 (1주 개발) — `agent-coding-1week-dev/`
5. ✅ N7 (1주 비개발) — `harness-agent-skills-1week/`
6. ✅ 공통 자산 — `_shared/`(glossary · safety-ethics · prompt-cookbook-dev)

> **6개 과정 전부 자료 제작 완료.** 각 과정은 소개서·운영안·슬라이드·실습(완성본/예시 포함)을 갖추며, 비개발은 N7 자산을, 개발은 `_shared/` 개발 쿡북과 D1·D3 랩을 공유·재사용합니다.
