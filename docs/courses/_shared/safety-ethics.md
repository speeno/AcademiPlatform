# 안전·윤리 가이드 (Safety & Ethics) — AX 교육 과정군 공용

> 모든 과정에서 동일하게 적용합니다. 강사는 실습 내내 아래를 강제합니다.

## 학습자 수칙
- ❌ **개인정보·기밀 금지**: 실명·주민번호·연락처·계약 원문·비밀번호·API 키 등을 AI에 붙여넣지 않는다.
- ✅ **샘플/가공 데이터만** 사용한다(익명화·일부 변형).
- ✅ **결과는 사람이 검증**한다. 특히 숫자·날짜·금액·법규·계약은 그대로 신뢰하지 않는다.
- ✅ 외부로 데이터를 전송하지 않는 범위(브라우저/로컬)에서 우선 실습한다.

## AI 결과의 한계 (반드시 인지)
- AI는 **그럴듯하지만 틀린 답**을 만들 수 있다(할루시네이션).
- "주어진 자료 안에서만 답하고, 없으면 '내용 없음'이라고 해줘"로 지어내기를 줄인다.
- 중요한 의사결정·대외 산출물은 **사람의 최종 확인**을 거친다.

## 강사 체크포인트
- 실데이터 사용 시도 발견 → 즉시 **샘플/가공본**으로 전환.
- 민감정보 입력 발견 → 즉시 중단·삭제 안내.
- 결과물 공개·배포 전 개인정보 포함 여부 점검.

## 사내 적용 시
- 사내 보안/IT 정책 확인(외부 도구 사용·데이터 반출 규정).
- 만든 도구의 **적용 범위와 한계**를 문서화해 오용을 막는다.
