# 사전 셋업 가이드 (개강 전 필수)

> 목표: **개강 첫날 바로 실습에 들어갈 수 있도록**, 계정·설치·접속을 미리 끝냅니다.
> 소요 시간: 약 30~40분. 막히면 아래 "문제 해결"과 담당자 연락처를 이용하세요.

---

## 0. 준비물 체크
- [ ] 노트북 (Windows 10+ 또는 macOS, 프로그램 **설치 권한** 있는 계정)
- [ ] 안정적인 인터넷 (사내망이면 아래 1번 방화벽 확인 필수)
- [ ] 업무 이메일 또는 개인 이메일 (계정 가입용)
- [ ] 풀고 싶은 업무 문제 1~2개 메모

## 1. 사내 네트워크/방화벽 확인 (가장 흔한 실패 지점)
브라우저에서 아래 두 주소가 정상적으로 열리는지 확인하세요. 차단되면 IT 담당자에게 허용 요청(과정 안내문 첨부).
- `https://claude.ai`
- `https://cursor.com`

## 2. Claude 계정 만들기
1. `https://claude.ai` 접속 → **가입(Sign up)**.
2. 이메일 인증 후 로그인.
3. 좌측에서 새 대화를 열어 "안녕"이라고 입력 → 응답이 오면 정상.
> 수업에서는 Claude를 ① 아이디어·문구·기획 도우미, ② Cursor와 함께 코드 생성·검토에 사용합니다.

## 3. Cursor 설치 & 로그인 (메인 도구)
1. `https://cursor.com` → **Download** → OS에 맞는 설치파일 내려받아 설치.
2. Cursor 실행 → **Sign in** (구글/깃허브/이메일 중 택1로 가입·로그인).
3. 첫 실행 시 안내(키맵 등)는 기본값으로 두고 넘어갑니다.
4. 화면 구성 익히기(상세는 Day 1에서):
   - **왼쪽**: 파일 탐색기(폴더·파일 목록)
   - **가운데**: 편집기(파일 내용)
   - **오른쪽/단축키**: **AI 채팅 / Agent(에이전트)** — 여기에 자연어로 시킵니다.

## 4. 작업 폴더 만들기
1. 바탕화면에 `ax-week` 라는 폴더를 새로 만듭니다.
2. Cursor에서 **File → Open Folder → `ax-week` 선택**.
3. 왼쪽 탐색기에 빈 폴더가 열리면 준비 완료.

## 5. 동작 확인 미니 테스트 (3분)
1. Cursor에서 AI 패널(Agent/Chat)을 엽니다. (단축키는 버전마다 다르며, 보통 우측 사이드바 또는 `Ctrl/Cmd + L`)
2. 다음을 그대로 입력해 보냅니다:
   ```
   이 폴더에 hello.txt 파일을 만들고, 그 안에 "셋업 완료!" 라고 한국어로 적어줘.
   ```
3. AI가 파일 생성을 제안하면 **수락(Accept/Apply)** 합니다.
4. 왼쪽 탐색기에 `hello.txt`가 생기고 내용이 들어 있으면 **셋업 성공** 🎉 (스크린샷을 남겨 두세요).

---

## 안전·윤리 (꼭 읽기)
- **개인정보·기밀을 붙여넣지 마세요.** 실명·주민번호·연락처·계약서 원문·비밀번호·API 키 등은 금지.
- 실습은 **샘플 데이터 또는 가공(익명화)된 데이터**로 진행합니다.
- AI의 결과는 **항상 사람이 검증**합니다. 특히 숫자·법규·계약 관련 내용은 그대로 신뢰하지 않습니다.

## 문제 해결 (Troubleshooting)
| 증상 | 해결 |
|---|---|
| 사이트가 안 열림 | 사내 방화벽 차단 가능 → IT에 `claude.ai`, `cursor.com` 허용 요청 |
| Cursor 설치 권한 없음 | 관리자 계정으로 설치 또는 IT 지원 요청 |
| 로그인 실패 | 다른 가입 방식(구글 등) 시도, 시크릿창 해제 |
| AI 패널이 안 보임 | 메뉴에서 보기(View)로 AI/Chat 패널 켜기, 또는 Cursor 최신 버전으로 업데이트 |

> 개강 전 5번 미니 테스트까지 통과하지 못하면, 첫날 30분 일찍 오셔서 도움을 받으세요.
