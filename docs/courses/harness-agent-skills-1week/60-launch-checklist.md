## B. 강사 / 운영 인력
- [ ] 메인 강사 1명 + (권장) 조교 1명 확보
- [ ] 강사가 `50-facilitator-guide.md` 숙지 + 랩 완성본 1회 실습
- [ ] 데모 데이 평가자(강사) 일정 확정

## C. 환경 / 기술
- [ ] 강의장: 대형 화면/빔, 콘센트, 팀 단위 섬 배치, 안정적 와이파이
- [ ] **방화벽 점검**: `claude.ai`, `cursor.com` 접근 가능(사내 진행 시 필수)
- [ ] 계정/비용 정책 확정(Claude·Cursor 무료/유료 여부, 사번/이메일 정책)
- [ ] 강사 PC에 완성본·슬라이드·백업 자료 준비

## D. 모집 / 등록
- [ ] 소개서(`01-syllabus.md`) 기반 모집 페이지/공지 게시
- [ ] 정원·일정·장소·가격 확정 및 신청 링크 연결
- [ ] 수료증 발급 절차/문구 확정(플랫폼 연계)
- [ ] 신청자에게 **사전 셋업 가이드 발송**(개강 D-3) + 완료 독려

## E. 개강 직전 (D-1)
- [ ] 사전 셋업 완료율 점검 → 미완료자 첫날 30분 일찍 안내
- [ ] 슬라이드/타임테이블 최종 확인
- [ ] 안전·윤리 안내문(개인정보 금지) 준비
- [ ] 데일리 체크인/회고 양식 준비

## F. 운영 중 (매일)
- [ ] 데일리 체크인 10분(어제/오늘/막힌 점)
- [ ] 일일 산출물 수합(루브릭 §3)
- [ ] 진도 격차 모니터링(빠른 팀/느린 팀 대응)

## G. 종료 후
- [ ] 평가/수료 판정(`40-assessment-rubric.md`) → 수료증 발급
- [ ] 만족도·난이도 설문
- [ ] 우수 결과물 사례 정리(동의 시) → 사내 확산
- [ ] 후속 과정 제안(1일/3일 심화, 팀 코칭)

## 최종 오픈 게이트 ✅
아래 4가지가 충족되면 오픈:
1. **자료(A) 완료** — ✅ 완료됨
2. **강사(B) 확보 + 숙지**
3. **환경(C) 점검** — 특히 방화벽/계정
4. **등록(D) 오픈** — 모집 페이지·신청 링크·수료증 연계

> B·C·D는 운영 측 준비 항목입니다. 자료(A)는 제작 완료 상태이므로, **운영 준비만 마치면 즉시 개강 가능**합니다.
