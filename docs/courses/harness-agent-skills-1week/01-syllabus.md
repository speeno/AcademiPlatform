# [공개용] 비개발자를 위한 1주 AX 실무 과정
## 하네스 & Agent Skills로 직접 만드는 바이브 코딩

> 코드를 못 짜도, 일주일 뒤 내 업무 문제를 푸는 **실제로 동작하는 도구**를 직접 만들어 갑니다.

---

### 이런 분께 권합니다
- 코딩 경험이 없거나 거의 없는 실무자(기획·마케팅·운영·HR·CS·디자인 등)
- "AI로 업무를 자동화하고 싶은데 어디서 시작할지 모르겠다"는 분
- 반복 업무·수작업 보고서·문의 응대 등 **풀고 싶은 실제 문제**가 있는 분

### 과정에서 얻는 것
- AI에게 일을 시키는 법(프롬프트)과 **바이브 코딩** 실전 감각
- **하네스(Cursor·Claude)** 로 AI가 직접 파일을 만들고 고치게 하는 능력
- 재사용 가능한 **Agent Skills(`SKILL.md`)** 와 기술 파일 작성법
- 막연한 불편을 **풀 수 있는 문제로 정의하고 분해**하는 사고법
- **팀 프로젝트 결과물 1개**(데모 가능한 실제 도구) + 사내 확산 계획

---

### 운영 정보
| 항목 | 내용 |
|---|---|
| 기간 | 5일 (1일 6시간, 총 30시간) |
| 방식 | 라이브 강의 + 팀 실습 중심(이론 30% / 실습·프로젝트 70%) |
| 정원 | 12~20명 (4~5인 팀제) |
| 도구 | Cursor(메인) + Claude — 계정/설치는 사전 셋업 가이드로 준비 |
| 수료 | 출석 80%↑ + 일일 산출물 + 팀 데모 → 수료증 발급 |

### 선수 조건 / 준비물
- 노트북(설치 권한), 안정적 인터넷
- Claude·Cursor 계정 (개강 전 [사전 셋업 가이드](./02-prerequisites-setup.md)로 완료)
- **본인 업무에서 풀고 싶은 문제 1~2개**를 적어 오기(프로젝트 소재)

---

### 5일 한눈에 보기
| 일자 | 주제 | 끝나면 할 수 있는 것 |
|---|---|---|
| Day 1 | AI로 일하기 & 첫 성공 | 자연어로 시켜서 동작하는 미니 도구 1개 완성 |
| Day 2 | 하네스 이해 & 에이전트처럼 일 시키기 | AI가 파일 생성·실행·수정하는 다단계 작업 수행 |
| Day 3 | Agent Skills & 기술 파일 만들기 | 나만의 `SKILL.md`·기술 파일 작성·호출 |
| Day 4 | 문제 정의 & 팀 프로젝트 빌드 | 문제 정의서·분해표 작성 + 빌드 착수 |
| Day 5 | 빌드 완료 & 데모 | 동작하는 팀 결과물 완성 + 시연 + 확산 계획 |

### 최종 산출물
각 팀이 **현업의 실제 문제 1개**를 정의하고, 바이브 코딩으로 **동작하는 결과물**을 만들어 마지막 날 데모합니다.
(예: 반복 문서 자동 생성기, 사내 FAQ/가이드 챗봇, 데이터 정리·리포트 자동화, 간단 신청/체크 웹폼, 회의록 요약·할 일 추출 도구 등)

---

### 자주 묻는 질문
- **코딩을 전혀 못 해도 되나요?** 네. 자연어로 의도를 설명하면 AI가 코드를 만들고, 우리는 "무엇을·왜"에 집중합니다.
- **결과물을 회사에서 계속 쓸 수 있나요?** 네. 만든 파일과 스킬을 그대로 가져가 확산하는 방법을 Day 5에서 다룹니다.
- **내 회사 데이터를 써도 되나요?** 민감정보·개인정보는 사용하지 않도록 안전 가이드를 제공하며, 실습은 샘플/가공 데이터로 진행합니다.
