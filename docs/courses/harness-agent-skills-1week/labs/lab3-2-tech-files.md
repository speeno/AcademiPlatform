# 랩 3-2 — 프로젝트 기술 파일 세트 (Day 3)

> 목표: 프로젝트를 굴러가게 하고 **AI에게 맥락을 주는** 기술 파일 세트를 만든다. 이게 Day 4 빌드의 출발점이 된다.
> 결과물: 팀 프로젝트 폴더 + `README.md` · `config.json` · `sample-data.csv` · `my-skills/.../SKILL.md`

## 왜 만드나
이 파일들이 곧 **AI에게 주는 맥락**입니다. 잘 갖출수록 "이 설명대로 만들어줘" 한마디로 품질이 올라갑니다.

## 진행 단계
**1단계 — 프로젝트 폴더**
```
ax-week 안에 team-project 폴더를 만들어줘.
```
**2단계 — README(프로젝트 설명서)**
```
team-project/README.md 를 만들어줘. 우리가 만들 것: (문제 정의 캔버스 1~2줄 요약). 입력/출력/실행 방법을 적어줘.
```
**3단계 — 설정 파일 & 샘플 데이터**
```
team-project/config.json 에 제목과 기본 옵션을 넣어줘.
team-project/sample-data.csv 에 익명화된 샘플 데이터 5줄을 만들어줘. (실데이터 금지)
```
**4단계 — 뼈대 생성(빌드 출발점)**
```
README.md 설명대로 이 프로젝트의 가장 작은 동작 버전(빈 화면+제목)을 만들어줘.
```

## 확인(검증)
- [ ] `README.md` 에 "무엇을/입력/출력/실행"이 있다
- [ ] `config.json` 이 열린다(형식 오류 없음)
- [ ] `sample-data.csv` 에 **가공된** 샘플만 있다
- [ ] 가장 작은 동작 버전이 화면에 뜬다

---

## ✅ 완성본 (예시 세트)
> 프로젝트 예: "주간 정산 엑셀 합치기 도우미" (팀 주제에 맞게 바꿔 쓰세요)

### `team-project/README.md`
```markdown
# 주간 정산 합치기 도우미

## 무엇을 만드나
운영팀이 매주 여러 엑셀에 흩어진 정산 내역을, 붙여넣기만 하면 하나의 표로 합쳐 주는 한 페이지 웹 도구.

## 입력 / 출력
- 입력: 표 형식 텍스트(쉼표 또는 탭 구분) 여러 개
- 출력: 합쳐진 표 + 합계 행

## 실행 방법
- index.html 을 브라우저로 연다. 입력칸에 붙여넣고 "합치기"를 누른다.

## 규칙
- 외부로 데이터를 전송하지 않는다(브라우저 안에서만 처리).
- 개인정보는 넣지 않는다.
```

### `team-project/config.json`
```json
{
  "title": "주간 정산 합치기 도우미",
  "columns": ["날짜", "항목", "금액"],
  "showTotal": true,
  "language": "ko"
}
```

### `team-project/sample-data.csv` (익명화 샘플)
```csv
날짜,항목,금액
2026-07-01,사무용품,12000
2026-07-02,교통비,8200
2026-07-03,식대,15000
2026-07-04,사무용품,4300
2026-07-05,교통비,9100
```

### 그리고 `my-skills/<업무>/SKILL.md`
- 랩 3-1에서 만든 스킬을 함께 둔다(예: 합친 결과를 보고서로 정리하는 스킬).

## AI가 이 파일들을 쓰게 하기
```
team-project/README.md 와 config.json 을 참고해서, sample-data.csv 같은 입력을 합쳐 표로 보여주는 index.html을 만들어줘. config의 columns와 showTotal을 반영해줘.
```

> 포인트: 사람이 README/설정만 잘 적어두면, **나머지는 하네스가 만든다.** 내일(Day 4) 이걸 본격적으로 키웁니다.
