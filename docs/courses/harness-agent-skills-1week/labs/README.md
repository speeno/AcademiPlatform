# 실습 랩 가이드 (Labs)

> 각 랩은 **따라하기(단계) + 예시 프롬프트 + 완성본(정답)** 으로 구성됩니다.
> **먼저 스스로 시도**하고, 막히면 "완성본"을 참고하세요. 완성본은 "이렇게 나오면 성공"의 기준입니다.

## 랩 진행 규칙
- **작게, 자주**: 한 번에 한 단계. 되면 다음.
- **항상 확인**: 변경은 수락(Apply) 후 **실제로 실행**해 본다.
- **오류는 친구**: 오류 메시지를 그대로 AI에 붙여 "원인 설명하고 고쳐줘".
- **안전**: 개인정보·기밀 금지, 샘플 데이터만.

## 결과물(HTML)을 화면에서 보는 법
- 만든 `*.html` 파일을 **더블클릭 → 브라우저로 열기**, 또는 Cursor에서 파일 우클릭 → "Reveal in Finder/Explorer" 후 브라우저로 열기.
- 수정 후에는 브라우저 **새로고침(F5)**.
