## 운영 핵심 팁
- **검증 강박**: 모든 AI 변경은 테스트·실행·diff 리뷰 후 머지(자동 수락 금지).
- **비밀 관리**: 토큰·`.env`·고객 데이터 금지. MCP 토큰은 안전하게 취급, 화면 공유 주의.
- **버전 변동성**: 모델/도구/MCP 설정은 공식 문서 기준으로 그때그때 확인(자료 표기는 개념 위주).
- **팀 표준화 관점**: "한 명이 잘 쓰기" → "팀 워크플로우로 고정"이 D3의 목표. 스킬셋·`CLAUDE.md`·자동화를 레포에 커밋.
- 안전·윤리: [safety-ethics](../_shared/safety-ethics.md) 상시 적용.
