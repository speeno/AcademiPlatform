# [D3] 에이전트 코딩 3Day — 개발자

> 1일 과정의 기본기를 넘어, **스킬 라이브러리·서브에이전트·MCP 연동·자동화 파이프라인**까지. 팀이 실제 레포에 에이전트 워크플로우를 적용·표준화합니다.

## 한 줄 소개
> "에이전트를 '잘 쓰는' 단계에서 '팀 워크플로우로 표준화'하는 단계로."

---

## 운영 정보
| 항목 | 내용 |
|---|---|
| 대상 | 현업 개발자(Git·터미널·테스트 경험) |
| 기간 | 3일 (1일 6시간, 총 18시간) |
| 정원 | 10~18명 (2~4인 팀제) |
| 도구 | Cursor + **Claude Code(CLI)** + Claude API/Agent SDK · MCP |
| 형태 | 강의 25% + 실습·팀 적용 75%(실제 레포) |
| 수료 | 출석 + 일일 산출물 + **팀 적용 결과(자동화/스킬) 데모** |

## 학습 성과
- 에이전트 코딩 멘탈 모델·플랜 워크플로우를 능숙히 운용한다.
- 재사용 가능한 **Agent Skill 라이브러리**와 `CLAUDE.md`를 팀 단위로 구축한다.
- **서브에이전트**로 작업을 위임·병렬화하고, **MCP**로 외부 도구/데이터를 연결한다.
- **자동화**(코드리뷰·테스트 생성·훅/슬래시 커맨드, CI 통합)를 설계·적용한다.
- 생성 코드의 **검증·보안·비밀관리** 기준을 적용한다.

## 3일 한눈에
| 일자 | 주제 | 끝나면 |
|---|---|---|
| Day 1 | 멘탈 모델 + 실전 워크플로우 + `CLAUDE.md` | 플랜→실행→검증 흐름 체득 |
| Day 2 | 스킬 라이브러리 · 서브에이전트 · MCP 연동 | 팀 스킬셋 + 외부 도구 연결 |
| Day 3 | 자동화(리뷰·테스트·훅·CI) + 팀 적용 | 동작하는 자동화 + 데모 |

## 사전 셋업
- [ ] Git·Node.js(LTS)·터미널, 테스트 러너 사용 경험
- [ ] **Cursor** + **Claude Code(CLI)** 설치·로그인(공식 설치 안내 기준)
- [ ] 실습 레포(샘플 제공 또는 본인 레포, 비밀정보 없음) + 가능하면 **GitHub 계정**(MCP·PR 실습용)
- [ ] (선택) MCP 연동 실습용 토큰은 **수업 중 안내**에 따라 준비(비밀정보 주의)
