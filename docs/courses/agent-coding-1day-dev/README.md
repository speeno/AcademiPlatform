# [D1] 에이전트 코딩 1Day — 개발자

> 이미 코드를 아는 분이 **하네스(Claude Code · Cursor)와 Agent Skills**로 개발 생산성을 즉시 끌어올립니다. 멘탈 모델 → 실전 워크플로우 → 커스텀 스킬을 하루에.

## 한 줄 소개
> "AI에게 코드를 받는 게 아니라, **에이전트를 운용**해 더 빠르고 안전하게 개발한다."

---

## 운영 정보
| 항목 | 내용 |
|---|---|
| 대상 | 현업 개발자(언어 무관, Git 사용 경험) |
| 기간 | 1일 (6시간) |
| 정원 | 10~20명 |
| 도구 | Cursor + **Claude Code(CLI)** + Claude(채팅) |
| 형태 | 강의 30% + 실습 70%(실제 레포 대상) |
| 수료 | 출석 + 실습(워크플로우 적용 + `SKILL.md`·`CLAUDE.md` 작성) 제출 |

## 학습 성과
- 에이전트 코딩의 멘탈 모델(하네스=도구·맥락·루프, 플랜 모드)을 설명한다.
- 스펙→플랜→실행 흐름으로 **버그 수정/기능 추가** 워크플로우를 수행한다.
- 레포에 맞는 **커스텀 Agent Skill(`SKILL.md`)** 과 **`CLAUDE.md`**(프로젝트 메모리)를 작성·적용한다.
- 컨텍스트 관리·서브에이전트·코드리뷰/테스트 자동화의 기초를 안다.
- AI 변경을 **diff·테스트로 검증**하는 안전 습관을 갖춘다.

## 사전 셋업 (개강 전)
- [ ] Git, Node.js(LTS) 설치, 터미널 사용 가능
- [ ] **Cursor** 설치·로그인
- [ ] **Claude Code(CLI)** 설치·로그인 (공식 설치 안내에 따름) — 사내망이면 `claude.ai`/관련 도메인 접근 확인
- [ ] 실습용 **샘플 레포** 클론(과정 제공) 또는 본인 사이드 레포 1개 준비(민감정보 없는 것)
- [ ] 동작 확인: 레포에서 에이전트에게 "이 프로젝트 구조를 요약해줘" → 정상 응답

> 모델 ID·설치 명령 등 변동 정보는 **수업 시점 공식 문서 기준**으로 안내(자료에 단정 표기하지 않음).

## 준비물
- 노트북(설치 권한), 인터넷, 샘플/사이드 레포(비밀정보 없음).

## 수료 기준
- 출석 + 실습 결과(워크플로우 적용 커밋/브랜치 + 작성한 `SKILL.md`·`CLAUDE.md`) 제출.
