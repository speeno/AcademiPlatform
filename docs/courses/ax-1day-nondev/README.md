# [N1] AX 입문 1Day — 비개발자

> 하루 만에 두려움을 없애고, **자연어로 시켜서 동작하는 도구**를 직접 만들어 봅니다. 하네스·Agent Skills를 "맛보기"로 경험하고, 내 업무 적용 아이디어까지.

## 한 줄 소개
> "코딩 몰라도, 오늘 하루로 'AI로 만든다'는 감을 잡는다."

## 운영 정보
| 항목 | 내용 |
|---|---|
| 대상 | 코딩 경험이 없는 실무자(전 직군) |
| 기간 | 1일 (6시간) |
| 정원 | 12~24명 |
| 도구 | Cursor(메인) + Claude(채팅) |
| 형태 | 강의 30% + 실습 70%(개인 실습 중심) |
| 수료 | 출석 + **동작하는 미니 도구 1개** 제출 |

## 학습 성과
- AX·바이브 코딩이 무엇인지 자기 말로 설명한다.
- AI에게 잘 시키는 법(명확한 지시 4요소)을 익힌다.
- **동작하는 미니 도구 1개**를 자연어 지시만으로 완성한다.
- 하네스·`SKILL.md`를 맛보고, 내 업무 적용 아이디어를 MVP 한 줄로 정리한다.

## 사전 셋업 (개강 전, 약 20분)
1. 사내망이면 `claude.ai`, `cursor.com` 접근 가능 여부 확인(차단 시 IT 요청).
2. **Claude** 가입(`claude.ai`) → 로그인.
3. **Cursor** 설치(`cursor.com`) → 로그인.
4. 바탕화면에 `ax-1day` 폴더 생성 → Cursor에서 **Open Folder**.
5. 동작 확인: AI 패널에 `이 폴더에 hello.txt 만들고 "준비완료"라고 적어줘` → 파일 생기면 OK.
> 자세한 절차/문제 해결은 [1주 과정 셋업 가이드](../harness-agent-skills-1week/02-prerequisites-setup.md)와 동일.

## 준비물
- 노트북(설치 권한), 인터넷, 풀고 싶은 업무 불편 1~2개 메모.

## 수료 기준
- 출석 + **랩 결과물(동작하는 미니 도구) 제출**.
- (선택) 내 업무 적용 아이디어 1줄 공유.
