# ISO/IEC 17024 AI 국제자격 과정 강의자료 — 제작 계획

> 대상 자격: **AI Prompt Engineer** / **AI Creator** (IQCS·AI GTC, ISO/IEC 17024 AI 부문)
> 두 과정 모두 ① 시험 합격을 위한 **기본 내용** + ② 첨부 교재 기반 **심화 내용**을 제작한다.
> 이 문서는 설계도(계획)이며, 승인 후 과정 폴더에 실제 자료를 채운다.

---

## 1. 자격·과정 개요
| 항목 | 내용 |
|---|---|
| 인증 체계 | ISO/IEC 17024 (AI 부문) · IQCS 발급 · AI GTC 교육 |

## 2. 시험 구조 → 강의 설계 정합 (매우 중요)
강의는 아래 시험 구조에 **역설계**해 맞춘다.
- **Pre-test**: O/X 10문항(6/10 합격) — 기본 소양. → "입문/용어 점검" 모듈로 커버.
- **1차 필기**: 40문항, 60점 이상(24/40) — 이론. → **기본 과정(교재 PART 기반)** 으로 커버.
- **2차 실기/주관식**: 주관식 20문항 또는 실기 제작물, 60점 이상. → **심화 과정(실습/실기)** 으로 커버.
- **출제 과목**(IQCS 기준): 인공지능의 이해 / 이론 / 활용 / 윤리 함양 / 활용 실제 / AI 도구.
> 따라서 **기본=필기 대비(교재 이론) + Pre-test**, **심화=실기 대비(실습·제작) + 윤리/활용 실제**로 이원화한다.

## 3. 교재 매핑 (확인 완료)
### 과정 A — Prompt Engineer 교재 `ai-iso-prompt.pdf` (122p, 12 PART)
PART1 AI 정의·구성요소 / PART2 AI 역사·이정표 / PART3 AI 작동원리(지도·비지도) / PART4 자연어처리(NLP) / PART5 강화학습 응용 / PART6 컴퓨터비전·멀티모달·AR·VR / PART7 어텐션 메커니즘 / PART8 GPT vs 전통 검색 / PART9 **페르소나 기반 프롬프팅·롤플레잉** / PART10 RLHF / PART11 딥페이크 / PART12 AI 채용 시스템.
> 성격: AI 이론 비중이 큼(필기 대비에 적합) + 프롬프팅 실전은 PART9 중심. → **심화에서 실전 프롬프트 엔지니어링을 대폭 보강**해야 함.

### 과정 B — AI Creator 교재 `ai-iso-creator.pdf` (94p)
PART1 딥러닝 원리(CNN·RNN·LSTM) / PART2 GANs / PART3 신경망 스타일 전이(NST) / PART4 AI 음악 제작(MIDI·Waveform) / PART5 AI 애니메이션·VR·메타버스·3D / (이후 이미지·영상 생성 등 계속).
> 성격: 생성형 콘텐츠의 **기술 원리**(필기 대비) 중심. → **심화에서 유형별 실제 제작 실습**(이미지·영상·음악·3D)을 보강.

### 보조 자료

- `Downloads/1. 자격증 가이드 AI 핵심 기술 트렌드.pdf` — 트렌드 보강.

## 4. 과정 A — AI Prompt Engineer (강의 설계)
### A-1. 기본(필기·Pre-test 대비) — 교재 PART 기반 이론
- 모듈을 교재 PART에 1:1 정렬. 각 모듈: 학습목표 · 핵심개념 · 용어 · 도식 · **예상문제(O/X·4지선다)**.
- 핵심 묶음: (1) AI 정의·분류·작동원리(PART1~3) (2) NLP·강화학습·비전/멀티모달(PART4~6) (3) 어텐션·GPT·RLHF(PART7,8,10) (4) 프롬프팅·페르소나(PART9) (5) 윤리·사회(딥페이크 PART11)·활용(AI채용 PART12).
### A-2. 심화(실기·주관식 대비) — 실전 프롬프트 엔지니어링
교재를 넘어 **실습 중심**으로 보강:
- 프롬프트 설계 원리(역할·맥락·예시·제약), 구조화 프롬프트, 단계적 사고/체이닝, 페르소나·롤플레잉(PART9 확장).
- 멀티모달 프롬프트, 검색결합(RAG) 개념, 출력형식 제어, 평가·반복 개선.
- **AI 윤리·안전**(할루시네이션·편향·개인정보·딥페이크 악용 방지) 실무 가이드.
- **캡스톤**: 실제 업무 시나리오용 프롬프트 세트/봇 설계 → 2차 실기 제출물 형식과 정렬.

## 5. 과정 B — AI Creator (강의 설계)
### B-1. 기본(필기·Pre-test 대비) — 교재 PART 기반 이론
- 딥러닝 원리(CNN/RNN/LSTM), GANs, 신경망 스타일 전이, AI 음악(MIDI/Waveform), 애니메이션·3D·VR·메타버스, (이미지·영상 생성). 각 모듈 예상문제 포함.
### B-2. 심화(실기·제작 대비) — 콘텐츠 유형별 실제 제작
- **이미지**: 텍스트→이미지 생성, 스타일 전이 실습, 편집/업스케일.
- **영상/애니메이션**: 텍스트→영상, 아바타·립싱크, 간단 3D/메타버스 콘텐츠.
- **음악/오디오**: 작곡·사운드 생성, TTS.
- 저작권·윤리(생성물 권리·딥페이크 책임), 품질 평가.
- **캡스톤 포트폴리오**: 유형별 제작물 모음 → 2차 실기 제출물과 정렬.

## 6. 각 과정 산출물 인벤토리 (제작 대상)
과정별 폴더(`iso-prompt-engineer/`, `iso-ai-creator/`)에 생성:
- `README.md` 소개서(자격·시험·수료·교재 안내)
- `curriculum-30강.md` — **30강 차시표**(기본 N강 + 심화 M강, 교재 PART 매핑)
- `instructor-guide/` — 강의안(차시별 운영안·핵심 스크립트)
- `slides/` — 브랜드 테마(academiq) 적용 슬라이드 → PDF/PPTX
- `labs/` — 심화 실습(따라하기 + 결과 예시)
- `glossary.md` — 시험 핵심 용어집
- `exams/` — **Pre-test·1차 필기·2차 실기 모의고사 + 해설**(기존 시험지 시드 활용)
  - ➕ **문제은행 등록**: 모의고사 문항을 플랫폼 **문제은행(`/admin/exam/questions`) DB에 등록**(과목·난이도·유형 메타 포함)해, 온라인 시험 출제에 재사용. 문서(`exams/`)와 DB를 함께 산출.
- `assessment-rubric.md` — 평가·합격 기준(시험 구조 정합)
- 공통 자산 재사용: `../_shared/`(safety-ethics, 슬라이드 테마)

## 7. 제작 로드맵 (과정별, 권장 순서: 프롬프트 → 크리에이터)
- **Phase 1 — 골격**: README + 30강 차시표 + 용어집(교재 정독 기반).
- **Phase 2 — 기본 강의**: 차시별 강의안 + 슬라이드(필기 대비, 교재 PART 정렬).
- **Phase 3 — 심화 실습**: labs(실기 대비) + 캡스톤 가이드.
- **Phase 4 — 평가**: Pre-test/1차/2차 모의고사 + 해설 + 루브릭, 슬라이드 PDF/PPTX 변환, 오픈 체크.
> 각 Phase 종료 시 확인받고 진행. 기본(필기)부터 완성 → 심화(실기) 보강 순.
