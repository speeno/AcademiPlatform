# [D7] 에이전트 코딩 1주 — 개발자 (심화 풀코스)

> 기본기부터 **멀티에이전트 오케스트레이션 · 컨텍스트 엔지니어링 · 관측/평가 · 사내 표준화**까지. 팀이 실제 레포에 에이전트 워크플로우를 설계·적용·표준화하고, 확산 거버넌스까지 만듭니다.

## 한 줄 소개
> "에이전트를 '쓰는' 단계를 넘어, 팀과 조직의 **개발 표준**으로 만든다."

---

## 운영 정보
| 항목 | 내용 |
|---|---|
| 대상 | 현업 개발자/테크리드(Git·테스트·CI 경험) |
| 기간 | 5일 (1일 6시간, 총 30시간) · 7일 일정 시 +확장 모듈 |
| 정원 | 8~16명 (2~4인 팀제) |
| 도구 | Cursor + **Claude Code(CLI)** + **Claude API/Agent SDK** · MCP |
| 형태 | 강의 25% + 실습·팀 적용 75%(실제 레포) |
| 수료 | 출석 + 일일 산출물 + **팀 표준화 결과(스킬셋·자동화·문서) 데모** |

## 학습 성과
- 에이전트 코딩 워크플로우(플랜→실행→검증)를 능숙히 운용한다.
- 재사용 **스킬 라이브러리**·`CLAUDE.md`·**MCP 연동**·**자동화(CI/훅)** 를 구축한다.
- **멀티에이전트/서브에이전트 오케스트레이션**으로 큰 작업을 분할·검증한다.
- **컨텍스트 엔지니어링**(무엇을 넣고 줄일지, 캐싱·압축·검색)으로 비용·품질을 최적화한다.
- **관측·평가**(토큰/성능 추적, 프롬프트·스킬 평가)와 **안전·거버넌스** 기준을 세운다.
- 팀/조직 적용을 위한 **표준·확산 계획**을 산출한다.

## 5일 한눈에
| 일자 | 주제 | 끝나면 |
|---|---|---|

| Day 2 | 스킬 라이브러리 + 서브에이전트 | 팀 스킬셋 + 위임/병렬 |
| Day 3 | MCP 연동 + 자동화(리뷰·테스트·훅·CI) | 외부 연결 + 검증 자동화 |
| Day 4 | 멀티에이전트 오케스트레이션 + 컨텍스트 엔지니어링 + 관측/평가 | 큰 작업 분할·측정 |
| Day 5 | 팀 표준화 프로젝트 완성 + 데모 + 확산 거버넌스 | 사내 표준 산출물 + 데모 |
