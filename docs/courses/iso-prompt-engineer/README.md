# AI Prompt Engineer — ISO/IEC 17024 국제자격 과정

> IQCS·AI GTC ISO/IEC 17024(AI 부문) **AI Prompt Engineer** 자격 대비 교육 과정.
> 교재 『AI 전문가 국제자격증 과정 — AI Prompt Engineer(프롬프트엔지니어과정 1)』(이현길·남동선, 맨도롱북스) 기반.

## 과정 목표
- 1차 **필기** 합격을 위한 생성형 AI 이론을 체계적으로 익힌다(기본).
- 2차 **실기/주관식** 합격을 위한 **실전 프롬프트 엔지니어링** 역량을 기른다(심화).
- 자격 취득을 넘어 업무에 바로 쓰는 프롬프트 설계 능력을 갖춘다.
