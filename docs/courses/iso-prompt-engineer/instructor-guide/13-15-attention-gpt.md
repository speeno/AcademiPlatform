## 강 13 — 어텐션 메커니즘 ① : 등장 배경 (PART7)
**학습목표**: 어텐션이 왜 필요한지(RNN 한계)와 기본 개념을 설명한다.
- **어텐션(Attention)**: 입력 중 **가장 관련 깊은 부분에 집중(가중치 부여)** 하는 기술. "어디에 집중할까"에 답.
  - 예: "나는 점심으로 햄버거를 먹었다. 그것은 맛있었다"에서 '그것'=햄버거를 파악.
- **RNN(순환 신경망)/LSTM**: 문장을 순차 처리, 이전 단어 맥락을 **히든 스테이트**에 압축 저장.
- **RNN의 한계**: 문장이 길수록 앞 정보가 희미해지는 **병목(Bottleneck)·장기 의존성(Long-term dependency)** 문제.
- 어텐션은 이 한계를 극복 → 문장 길이와 무관하게 모든 단어 관계에 접근.
**예상문제(O/X)**: RNN은 문장이 길어질수록 앞 단어 정보가 잘 보존된다. → **X** (병목/장기의존성 문제).

## 강 14 — 어텐션 메커니즘 ② : 셀프어텐션과 혁신 (PART7)
**학습목표**: 셀프어텐션의 Q/K/V와 어텐션이 가져온 혁신을 설명한다.
- **셀프어텐션(Self-Attention)**: **한 문장 내** 각 단어와 다른 모든 단어의 관계를 동시에 파악. 트랜스포머의 심장.
- **3개 벡터**(도서관 비유):
  - **쿼리(Query)**: "내가 찾는 것?" (현재 단어가 던지는 질문)
  - **키(Key)**: "나는 이런 정보를 가졌다" (분류 라벨)
  - **값(Value)**: 실제 내용물.
  - 쿼리·키 비교 → **어텐션 스코어** → 값에 가중치 → 가중 평균이 단어의 새 표현.
- **어텐션이 가져온 혁신**:
  1. **병렬 처리**(GPU 활용) → 학습 속도 획기적 단축.
  2. **장기 의존성 해결** → 긴 문장 문맥 손실 극복.
  3. **거대 언어모델(LLM) 탄생** → GPT·BERT의 기반.
**예상문제(4지선다)**: 셀프어텐션에서 '현재 단어가 던지는 질문' 역할의 벡터는? ①키 ②값 ③**쿼리** ④토큰 → **③**.

## 강 15 — GPT vs 전통 검색 엔진 (PART8)
**학습목표**: 전통 검색과 GPT의 차이·장단점을 비교한다.
- **패러다임 차이**: 전통 검색=**키워드·링크로 문서 연결** / GPT=**문맥 이해로 새 정보 창조**.
- **전통 검색 장점(4)**: ①출처 명확 ②다양한 관점 ③정보 최신성·실시간 ④탐색 통제권.
- **전통 검색 한계(4)**: ①정제된 답 미제공(비효율) ②키워드-문맥 불일치 ③정보 과부하 ④창의성 부족.
- **GPT 문맥이해 장점**: 즉각·맞춤 답변 / 종합·논리적 설명 / 창의적 콘텐츠 생성 / 멀티모달(예: GPT-4o 이미지 분석).
- **GPT 한계**: 출처 불명확, 학습 시점 한계, 환각 가능 → 검증 필요.
**예상문제(O/X)**: 출처를 명확히 제시하는 데는 전통 검색이 GPT보다 유리하다. → **O**.
