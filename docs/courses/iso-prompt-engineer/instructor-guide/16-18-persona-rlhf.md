## 강 16 — 페르소나 기반 프롬프팅 (PART9)
**학습목표**: 페르소나 프롬프팅의 정의·목적·활용법을 설명하고 예시를 만든다.
- **정의**: AI에게 특정 인물의 성격·직업·전문성·관점·지식 수준을 **명시적으로 부여**해 답변 스타일·내용을 제어. ('페르소나'=가면)
- **목적**: 답변을 의도에 정확히 맞추고 **'질'을 높임**(전문가 깊이 / 초보자 눈높이).
- **활용법**: 프롬프트 앞에 "너는 이제부터 ~ 전문가야", "~의 관점에서 답변해 줘".
- **예시**:
  - 전문가: "너는 스타트업 전문 경영 컨설턴트야. 예상 시장 리스크와 해결책을 제시해 줘."
  - 교육자: "너는 초등 5학년에게 과학을 가르치는 선생님이야. '지구 온난화'를 쉽게 설명해 줘."
  - 인물: "너는 소크라테스야. '행복이란?'에 답해 줘."
  - 마케터: "너는 MZ세대 타깃 디지털 마케팅 전문가야. 숏폼 마케팅 전략을 세워 줘."
**예상문제(O/X)**: 페르소나 프롬프팅은 주로 답변의 '스타일·전문성'을 제어한다. → **O**.

## 강 17 — 롤플레잉 & 페르소나와의 차이 (PART9)
**학습목표**: 롤플레잉을 정의하고 페르소나 프롬프팅과 비교한다.
- **정의**: AI와 사용자가 **각자 역할**을 맡아 가상 상호작용을 **연속적으로** 진행. '대화 흐름·상황 전개'에 집중.
- **활용**: "우리는 이제부터 ~ 역할을 맡아볼 거야"로 양쪽에 역할 부여.
- **예시**: 면접 시뮬레이션, 고객 응대 연습, 스토리텔링(마스터/모험가), 가격 협상.
- **페르소나 vs 롤플레잉(표)**:
  | 구분 | 페르소나 | 롤플레잉 |
  |---|---|---|
  | 주요 목적 | 답변 스타일·전문성 | 상황 시뮬레이션·대화 전개 |
  | 상호작용 | 단일 질문 맞춤 답변 | 연속 대화·상황 가정 |
  | 역할 부여 | 주로 AI에게만 | AI·사용자 모두 |
  | 공통점 | AI에 특정 역할 부여로 상호작용의 질↑ | (동일) |
**예상문제(4지선다)**: AI와 사용자 모두에게 역할을 부여해 연속 대화를 진행하는 기법은? ①페르소나 ②**롤플레잉** ③요약 ④체이닝 → **②**.

## 강 18 — RLHF (인간 피드백 강화학습) (PART10)
**학습목표**: RLHF의 3단계와 GPT 응답 개선에서의 역할을 설명한다.
- **정의**: 강화학습의 한 종류로 **인간 피드백**을 학습에 추가 → LLM을 '예측기'에서 '인간 가치를 이해하는 비서'로 진화.
- **3단계(암기)**:
  1. **지도 미세조정(SFT)**: 고품질 질문-답변 쌍으로 형식·스타일 학습(아직 '왜 좋은지'는 모름).
  2. **보상 모델(Reward Model)**: 인간이 여러 답변에 순위 부여 → '인간 선호'를 점수로 예측하는 모델(RLHF의 핵심, '가치관' 학습).
  3. **강화학습(PPO)**: 보상 점수를 높이는 방향으로 GPT를 최종 파인튜닝.
- **결정적 역할**: 안전성·윤리(유해 답변 억제), 미묘한 맥락 파악, **환각 제어**, 창의·유용성 품질↑ → '인간 중심 AI'.
**예상문제(4지선다)**: RLHF 2단계에서 만드는 것은? ①SFT 모델 ②**보상 모델** ③토크나이저 ④임베딩 → **②**.
