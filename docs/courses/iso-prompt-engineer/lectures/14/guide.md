# 강 14 — 어텐션 메커니즘 ② : 셀프어텐션과 혁신

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 셀프어텐션의 Q/K/V와 어텐션이 가져온 혁신을 설명한다.
- **셀프어텐션(Self-Attention)**: **한 문장 내** 각 단어와 다른 모든 단어의 관계를 동시에 파악. 트랜스포머의 심장.
- **3개 벡터**(도서관 비유):
  - **쿼리(Query)**: "내가 찾는 것?" (현재 단어가 던지는 질문)
  - **키(Key)**: "나는 이런 정보를 가졌다" (분류 라벨)
  - **값(Value)**: 실제 내용물.
  - 쿼리·키 비교 → **어텐션 스코어** → 값에 가중치 → 가중 평균이 단어의 새 표현.
- **어텐션이 가져온 혁신**:
  1. **병렬 처리**(GPU 활용) → 학습 속도 획기적 단축.
  2. **장기 의존성 해결** → 긴 문장 문맥 손실 극복.
  3. **거대 언어모델(LLM) 탄생** → GPT·BERT의 기반.
**예상문제(4지선다)**: 셀프어텐션에서 '현재 단어가 던지는 질문' 역할의 벡터는? ①키 ②값 ③**쿼리** ④토큰 → **③**.
