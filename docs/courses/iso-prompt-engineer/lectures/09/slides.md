# 강 09
## 자연어처리(NLP) ② — 감정·NER·QA·토픽·교정·음성
AI Prompt Engineer · ISO/IEC 17024 — 필기 대비

> "AI가 의미를 '뽑아내고' 답을 '찾아주는' 법."

---

## 학습 목표
1. NLP 주요 응용을 **구분**한다
2. **NER·QA·감정분석·토픽모델링** 을 설명한다
3. 음성 인식 + 자연어 생성의 흐름을 안다

---

## 감정 분석 (Sentiment)
- 텍스트의 감정·태도(긍정/부정/중립) 분석
- 세부 감정(만족·불만·분노)까지 발전
- 활용: 리뷰·여론 분석, 제품 개선

---

## 정보 추출 & NER
- **정보추출(IE)**: 비정형 텍스트 → 구조화 데이터
- **명명 개체 인식(NER)**: 사람·장소·날짜·기관·금액 식별
  - 예: "2024년 5월 10일, 김철수, 서울시청" → 날짜·인명·장소

---

## 질의응답 (QA)
- 문서 나열이 아니라 **정확한 답** 제시
- 예: "에베레스트 높이?" → "8,848.86m"
- 지식베이스/백과 기반으로 답을 찾아 제공

---

## 토픽 모델링
- 방대한 문서에서 **숨은 주제** 자동 발견
- 각 문서가 어떤 주제·키워드로 구성되는지 분류
- 예: 게시물 → 정치·스포츠·생활 자동 분류

---

## 맞춤법·문법 교정
- 오타·문법·구두점 오류 자동 감지·수정
- NLP 기본기가 집약된 응용
- 예: 워드프로세서 맞춤법 검사, 작문 보조

---

## 음성 인식 + 자연어 생성
- **STT**(음성→텍스트) + **NLG**(텍스트 생성) 결합
- 음성 → 텍스트 변환 → 의미 파악(NLP) → 응답 → 음성
- 예: 시리, 빅스비, 구글 어시스턴트

---

## NLP 10대 응용 한눈에
- 번역 · 챗봇 · 요약 · **감정분석** · 정보추출
- **NER** · **QA** · 토픽모델링 · 맞춤법교정 · 음성↔텍스트

---

## 흔한 오해 (시험 주의)
- "QA는 관련 문서를 나열만 한다" → ❌ (정확한 답 제시)
- "NER은 문장을 요약한다" → ❌ (개체 식별)
