# 강 09 — 자연어처리(NLP) ②

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 감정분석·정보추출·NER·QA·토픽모델링·교정·음성 변환을 구분한다.
- **감정분석(Sentiment)**: 긍정/부정/중립 + 세부 감정(만족·불만·분노). 리뷰·여론 분석.
- **정보추출(IE)**: 비정형 텍스트→구조화 데이터(NER 포함).
- **명명 개체 인식(NER)**: 사람·장소·날짜·기관·금액 등 고유 개체 식별·분류.
- **질의응답(QA)**: 문서 나열이 아닌 **정확한 답** 제시(예: "에베레스트 높이?"→수치).
- **토픽 모델링**: 문서 집합의 숨은 주제 자동 발견.
- **맞춤법·문법 교정**: 오타·문법·구두점 자동 감지·수정.
- **음성인식(STT) + 자연어생성(NLG)**: 음성→텍스트→이해→응답 생성→음성(예: 시리·빅스비·구글 어시스턴트).
**예상문제(4지선다)**: "2024년 5월 10일, 김철수, 서울시청"에서 개체를 식별하는 기술은? ①요약 ②감정분석 ③**NER** ④번역 → **③**.
