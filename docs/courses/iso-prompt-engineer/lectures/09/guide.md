## 전개 (슬라이드별 강의 포인트)
### S3~4 감정·NER
- 감정분석(긍/부정·세부), NER(개체 식별) — "서울시청=장소" 예시로 각인.
### S5~6 QA·토픽
- QA=정확한 답(문서 나열 아님), 토픽모델링=숨은 주제 발견.
### S7~8 교정·음성
- 맞춤법 교정, STT+NLG 결합(시리·빅스비 흐름).
### S9 10대 응용 정리

## 예상문제 + 해설
1. **(4지선다)** 개체(인명·장소·날짜)를 식별하는 기술은? → **NER**.
2. **(O/X)** 토픽 모델링은 문서의 숨은 주제를 자동 발견한다. → **O**.
3. **(단답)** 음성을 텍스트로 바꾸는 기술의 약어는? → **STT**.
> 해설 포인트: QA=정확한 답, NER=개체 식별, 토픽=주제 발견.
