# 강 13 — 어텐션 메커니즘 ① : 등장 배경

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 어텐션이 왜 필요한지(RNN 한계)와 기본 개념을 설명한다.
- **어텐션(Attention)**: 입력 중 **가장 관련 깊은 부분에 집중(가중치 부여)** 하는 기술. "어디에 집중할까"에 답.
  - 예: "나는 점심으로 햄버거를 먹었다. 그것은 맛있었다"에서 '그것'=햄버거를 파악.
- **RNN(순환 신경망)/LSTM**: 문장을 순차 처리, 이전 단어 맥락을 **히든 스테이트**에 압축 저장.
- **RNN의 한계**: 문장이 길수록 앞 정보가 희미해지는 **병목(Bottleneck)·장기 의존성(Long-term dependency)** 문제.
- 어텐션은 이 한계를 극복 → 문장 길이와 무관하게 모든 단어 관계에 접근.
**예상문제(O/X)**: RNN은 문장이 길어질수록 앞 단어 정보가 잘 보존된다. → **X** (병목/장기의존성 문제).
