# 강 30 — 2차 실기 모의 + 리뷰 · 강의 스크립트

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지 (강사용 운영안) · 2차 실기 대비

## 강의 개요
- **시간**: 약 90분 (작성 70분 + 리뷰 20분)
- **목표**: 주관식·실기 실전 + 루브릭 자가진단
- **선수**: 강23~29 / **산출물**: 채점 결과·재작성본
- **자료**: `questions.md`(주관식5+실기2, 모범답안·루브릭)

## 운영 절차
1. **주관식(30분)**: 5문항 서술.
2. **실기(40분)**: 회의록 봇·규정 Q&A 프롬프트 작성·실행.
3. **채점(10분)**: 모범답안·루브릭으로 자가 채점.
4. **리뷰(10분)**: 감점 포인트 피드백 + 약점 재작성.

## 합격 기준
- 주관식 50 + 실기 50 = 100점, **60점 이상**.

## 실기 채점 핵심 (강의 포인트)
- **구조**: 시스템+핵심+형식 분리.
- **4요소**: 역할·맥락·예시·제약.
- **안전**: 근거 한정·미존재 처리·개인정보·검증.
- **완결성**: 요구사항 모두 충족.

## 자주 감점 (강조)
- 출력 형식 미지정 / "근거 없으면" 처리 누락 / 역할·제약 누락 / 개인정보·검증 무시.
