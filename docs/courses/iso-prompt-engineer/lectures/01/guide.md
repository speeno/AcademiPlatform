# 강 01 — OT & AI 기본 소양

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 과정·시험 구조를 이해하고, 생성형 AI의 큰 그림과 Pre-test 핵심 포인트를 잡는다.
- 과정 안내: 30강 구성(기본 22 + 심화 8), 시험 3단계(Pre-test→1차 필기→2차 실기).
- 전통적/분석적 AI(패턴 인식) vs **생성형 AI(새 콘텐츠 창조)** 의 차이.
- 생성형 AI 활용 범위: 텍스트·이미지·영상·음악·코드·업무 자동화.
- Pre-test 빈출(O/X): "AI=인간 지능 모방", "생성형 AI는 새 콘텐츠를 만든다", "딥러닝은 기계학습의 한 분야다".
**예상문제(O/X)**: 생성형 AI는 기존 데이터를 분석만 한다. → **X** (새 콘텐츠를 생성).
