# 강 06 — AI 작동원리 ① 지도학습 심화 · 강의 스크립트

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지 (강사용 운영안) · 교재 PART3

## 강의 개요
- **시간**: 약 25분 (슬라이드 12장)
- **목표**: 지도학습 작동·분류/회귀·산업활용·성능지표·데이터비용
- **선수**: 강04 / **산출물**: 분류 vs 회귀, 성능지표 구분

## 시간 배분
| 구간 | 시간 | 슬라이드 |
|---|---|---|
| 도입·작동 | 5분 | 3~4 |
| 분류/회귀 | 4분 | 5 |
| 산업 활용 | 7분 | 6~7 |
| 지표·비용 | 6분 | 8~9 |
| 오해·정리·퀴즈 | 3분 | 10~11 |

## 도입 (멘트)
> "지도학습은 '정답지를 보며 공부하는 학생'입니다. 입력과 정답의 관계를 배워서, 새 문제의 정답을 맞히죠. 산업에서 가장 많이 쓰이는 방법이라 활용 사례와 평가지표가 시험에 자주 나옵니다."

## 전개 (슬라이드별 강의 포인트)
### S4 작동
- 입력-정답 반복 학습 → **오차를 줄이는** 방향. 일반화 강조.
### S5 분류/회귀
- 표로 대비, "연속값=회귀"를 반복(빈출).
### S6~7 산업 활용
- 금융·의료·유통·제조·자율주행 사례를 학습자 직군과 연결.
### S8~9 지표·비용
- 정확도/정밀도/재현율/MSE 구분. 라벨링 비용이 크다는 점(의료영상 예).

## 활동 / 토론 (3분)
- "내 분야의 지도학습 문제를 분류/회귀로 분류하고 평가지표 정하기."

## 정리 (멘트)
> "지도학습은 정답과 오차로 배우고, 객관적 평가가 가능합니다. 다음 강은 비지도·강화와 세 학습의 차이를 표로 마무리합니다."

## 예상문제 + 해설
1. **(4지선다)** 연속값을 예측하는 과제는? → **회귀**.
2. **(O/X)** 지도학습은 테스트 데이터로 객관적 평가가 가능하다. → **O**.
3. **(단답)** 회귀의 대표 오차 지표는? → **MSE(평균제곱오차)**.
> 해설 포인트: 분류 vs 회귀, 분류 지표(정밀도·재현율) vs 회귀 지표(MSE), 라벨 비용.
