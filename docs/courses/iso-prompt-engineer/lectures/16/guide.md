- **시간**: 약 23분 (슬라이드 9장)
- **목표**: 페르소나 개념 + 역할 부여 효과 + 기본 구조
- **선수**: 강15 / **산출물**: 페르소나 프롬프트 1개 작성

## 도입 (멘트)
> "'건강에 좋은 식단?'을 의사·요리사·영양사에게 물으면 답이 다 다르죠. AI도 마찬가지입니다. '너는 누구야'를 정해주면 답의 관점과 전문성이 달라집니다. 이게 페르소나입니다."

## 전개 (슬라이드별 강의 포인트)
### S4 개념
- 역할·정체성 부여 → 관점/어조/전문성 조정.
### S5 효과
- 지식 영역 활성화·일관된 어조·방향성.
### S6 구조
- 역할+대상+요청+제약 4요소. 실습용 템플릿.
### S7 좋은/나쁜 예
- "식단 알려줘" vs 역할·대상·형식 명시 비교.

## 활동 / 실습 (4분)
- 각자 자기 직무 관련 페르소나 프롬프트 1개 작성 → 2명 발표.

## 정리 (멘트)
> "페르소나는 역할 부여로 답을 조정하는 기법입니다. 다음 강은 이를 롤플레잉으로 확장합니다."

## 예상문제 + 해설
1. **(4지선다)** 페르소나 프롬프팅의 핵심은? → **역할 부여**.
2. **(O/X)** 역할 부여로 답변 어조·전문성이 달라진다. → **O**.
> 해설 포인트: 페르소나=역할, 구조=역할+대상+요청+제약.
