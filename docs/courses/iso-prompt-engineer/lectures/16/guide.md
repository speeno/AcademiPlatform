# 강 16 — 페르소나 기반 프롬프팅

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 페르소나 프롬프팅의 정의·목적·활용법을 설명하고 예시를 만든다.
- **정의**: AI에게 특정 인물의 성격·직업·전문성·관점·지식 수준을 **명시적으로 부여**해 답변 스타일·내용을 제어. ('페르소나'=가면)
- **목적**: 답변을 의도에 정확히 맞추고 **'질'을 높임**(전문가 깊이 / 초보자 눈높이).
- **활용법**: 프롬프트 앞에 "너는 이제부터 ~ 전문가야", "~의 관점에서 답변해 줘".
- **예시**:
  - 전문가: "너는 스타트업 전문 경영 컨설턴트야. 예상 시장 리스크와 해결책을 제시해 줘."
  - 교육자: "너는 초등 5학년에게 과학을 가르치는 선생님이야. '지구 온난화'를 쉽게 설명해 줘."
  - 인물: "너는 소크라테스야. '행복이란?'에 답해 줘."
  - 마케터: "너는 MZ세대 타깃 디지털 마케팅 전문가야. 숏폼 마케팅 전략을 세워 줘."
**예상문제(O/X)**: 페르소나 프롬프팅은 주로 답변의 '스타일·전문성'을 제어한다. → **O**.
