## 전개 (슬라이드별 강의 포인트)
### S4 전통 검색
- 키워드 매칭 + 링크 순위(페이지랭크). 사용자가 종합.
### S5~6 GPT·비교표
- 문맥 이해·생성·대화형. 비교표로 각인.
### S7 장단점
- 장점=가공된 답, 한계=**할루시네이션**·학습 시점. **빈출**.
### S8 RAG 예고
- 검색+LLM 결합=RAG, 강26 연결.

## 활동 / 토론 (3분)
- "검색이 더 나은 상황 vs GPT가 더 나은 상황 각 1개."

## 정리 (멘트)
> "GPT는 강력하지만 할루시네이션을 검증해야 합니다. 다음 강부터 페르소나 기반 프롬프팅으로 들어갑니다."
