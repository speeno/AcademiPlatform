# 강 10 — 강화학습과 응용 분야

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 강화학습 구성요소와 5대 응용을 설명한다.
- **구성요소(암기)**: 에이전트(Agent)·환경(Environment)·상태(State)·행동(Action)·보상(Reward)·정책(Policy).
- **목표**: 누적 보상 최대화 정책 학습(정답 데이터 없거나 경우의 수가 너무 많을 때 강력).
- **보상 설계 예(자율주행)**: 목적지 근접 +100 / 충돌 −500 / 신호위반 −100. 안전·연비·최단경로에 긍정 보상.
- **5대 응용**: ①자율주행 ②로봇 공학·제어 ③게임 AI(알파고) ④금융 투자·거래 ⑤자원 관리·최적화(예: 구글 데이터센터 냉방 전력 절감).
**예상문제(O/X)**: 강화학습의 목표는 즉시 보상 1회를 최대화하는 것이다. → **X** (장기 누적 보상).
