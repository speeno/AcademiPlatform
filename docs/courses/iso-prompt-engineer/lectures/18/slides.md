# 강 18
## RLHF — 인간 피드백 기반 강화학습
AI Prompt Engineer · ISO/IEC 17024 — 필기 대비

> "AI를 '사람이 원하는 답'에 맞추는 비결."

---

## 학습 목표
1. **RLHF** 의 개념을 안다
2. **3단계** 학습 과정을 이해한다
3. RLHF가 **왜 중요** 한지 설명한다

---

## 도입 — 똑똑함 ≠ 친절함
- 거대 모델은 지식은 많지만
- 무례하거나·위험하거나·엉뚱할 수 있음
- **사람의 선호** 에 맞추는 과정이 필요

---

## RLHF란?
- **Reinforcement Learning from Human Feedback**
- 인간 피드백으로 모델의 답을 **정렬(align)**
- ChatGPT가 자연스러워진 핵심 기술

---

## 1단계 — 지도학습 미세조정(SFT)
- 사람이 작성한 **모범 답변** 으로 학습
- 기본적인 대화·지시 따르기 능력 확보
- "이렇게 답하는 거야" 시범

---

## 2단계 — 보상 모델 학습
- 한 질문에 여러 답변 생성
- 사람이 **더 좋은 답에 순위** 매김
- 이 선호를 학습한 **보상 모델** 생성

---

## 3단계 — 강화학습 최적화
- 보상 모델을 **보상 신호** 로 사용
- 모델이 **높은 점수 답** 을 내도록 강화학습
- 강10의 강화학습 원리 적용

---

## RLHF의 효과
- 더 **유용·정직·무해** 한 답변
- 지시를 잘 따르고 어조가 자연스러움
- 단, 사람 평가자의 **편향** 도 반영될 수 있음

---

## 흔한 오해 (시험 주의)
- "RLHF는 1단계로 끝난다" → ❌ (SFT→보상모델→강화학습 3단계)
- "RLHF는 모델 지식을 늘린다" → ❌ (선호·정렬을 조정)

---
