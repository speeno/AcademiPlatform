# 강 18 — RLHF (인간 피드백 강화학습)

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: RLHF의 3단계와 GPT 응답 개선에서의 역할을 설명한다.
- **정의**: 강화학습의 한 종류로 **인간 피드백**을 학습에 추가 → LLM을 '예측기'에서 '인간 가치를 이해하는 비서'로 진화.
- **3단계(암기)**:
  1. **지도 미세조정(SFT)**: 고품질 질문-답변 쌍으로 형식·스타일 학습(아직 '왜 좋은지'는 모름).
  2. **보상 모델(Reward Model)**: 인간이 여러 답변에 순위 부여 → '인간 선호'를 점수로 예측하는 모델(RLHF의 핵심, '가치관' 학습).
  3. **강화학습(PPO)**: 보상 점수를 높이는 방향으로 GPT를 최종 파인튜닝.
- **결정적 역할**: 안전성·윤리(유해 답변 억제), 미묘한 맥락 파악, **환각 제어**, 창의·유용성 품질↑ → '인간 중심 AI'.
**예상문제(4지선다)**: RLHF 2단계에서 만드는 것은? ①SFT 모델 ②**보상 모델** ③토크나이저 ④임베딩 → **②**.
