# 강 20 — AI 채용 시스템 적용과 훈련

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: AI 채용 사례와 편향 문제·올바른 훈련법을 설명한다.
- **적용 단계**: 주로 서류 전형·1차 면접에 도입(효율·객관 데이터).
- **국내 사례**: LG(자소서 표절·키워드 분석), 포스코·SK C&C(서류 선별·후보 추천), 마이다스 **inAIR**(답변·목소리·표정·시선 분석 AI 면접).
- **해외 사례**: 유니레버(게임+비디오 면접, 시간·비용 절감·다양성↑), 보다폰(대량 지원서 분석), **아마존(편향으로 폐기)** — 10년 남성 위주 데이터 학습 → '여성' 키워드 감점, 2017 폐기.
- **편향 핵심 교훈**: 학습 데이터 다양성 결여 → 편향 재생산. "공정한 AI"는 키워드 필터링 이상의 과제.
- **올바른 훈련 3단계**: ①데이터 수집·전처리(다양성 확보·민감정보 제거·증강) ②모델 훈련·검증(공정성 지표·적대적 디바이어싱·설명가능성 XAI) ③사후 관리(Human-in-the-loop·성과 피드백 루프).
**예상문제(O/X)**: 아마존 AI 채용 시스템은 학습 데이터 편향으로 여성에 불리해 폐기됐다. → **O**.
