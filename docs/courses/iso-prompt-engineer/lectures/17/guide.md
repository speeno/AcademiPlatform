# 강 17 — 롤플레잉 & 페르소나와의 차이

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 롤플레잉을 정의하고 페르소나 프롬프팅과 비교한다.
- **정의**: AI와 사용자가 **각자 역할**을 맡아 가상 상호작용을 **연속적으로** 진행. '대화 흐름·상황 전개'에 집중.
- **활용**: "우리는 이제부터 ~ 역할을 맡아볼 거야"로 양쪽에 역할 부여.
- **예시**: 면접 시뮬레이션, 고객 응대 연습, 스토리텔링(마스터/모험가), 가격 협상.
- **페르소나 vs 롤플레잉(표)**:
  | 구분 | 페르소나 | 롤플레잉 |
  |---|---|---|
  | 주요 목적 | 답변 스타일·전문성 | 상황 시뮬레이션·대화 전개 |
  | 상호작용 | 단일 질문 맞춤 답변 | 연속 대화·상황 가정 |
  | 역할 부여 | 주로 AI에게만 | AI·사용자 모두 |
  | 공통점 | AI에 특정 역할 부여로 상호작용의 질↑ | (동일) |
**예상문제(4지선다)**: AI와 사용자 모두에게 역할을 부여해 연속 대화를 진행하는 기법은? ①페르소나 ②**롤플레잉** ③요약 ④체이닝 → **②**.
