# 강 17 — 페르소나·롤플레잉 ② · 강의 스크립트

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지 (강사용 운영안) · 교재 PART9

## 강의 개요
- **시간**: 약 24분 (슬라이드 10장)
- **목표**: 롤플레잉 패턴 + 3대 활용 + 팁
- **선수**: 강16 / **산출물**: 롤플레잉 프롬프트 1개 작성

## 시간 배분
| 구간 | 시간 | 슬라이드 |
|---|---|---|
| 도입·패턴 | 7분 | 3~4 |
| 3대 활용 | 9분 | 5~7 |
| 팁 | 3분 | 8 |
| 오해·정리·퀴즈 | 4분 | 9~10 |

## 도입 (멘트)
> "앞 강의 '너는 영양사다'에서 한 걸음 더 — '화난 고객을 응대하는 상황'을 더하면 AI가 상황극을 합니다. 역할에 상황과 목표를 더한 게 롤플레잉입니다."

## 전개 (슬라이드별 강의 포인트)
### S4 패턴
- 역할+상황+상대+목표 4요소. 콜센터 예.
### S5~7 활용
- 면접/상담 연습, 글쓰기/창작, 업무 시뮬레이션.
### S8 팁
- 구체성, 대화 규칙("3문장"), 역할 재강조.

## 활동 / 실습 (4분)
- 자기 업무 시나리오로 롤플레잉 프롬프트 작성 → 짝과 교환 테스트.

## 정리 (멘트)
> "롤플레잉은 역할에 상황·목표를 더해 연습·창작·시뮬레이션에 강력합니다. 다음 강은 모델을 인간 선호에 맞추는 RLHF입니다."

## 예상문제 + 해설
1. **(4지선다)** 롤플레잉이 페르소나에 추가하는 요소는? → **상황·목표**.
2. **(O/X)** 롤플레잉은 면접·상담 연습에 활용 가능하다. → **O**.
> 해설 포인트: 롤플레잉=역할+상황+상대+목표.
