# 강 07 — AI 작동원리 ② 비지도·강화 & 차이

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지

**학습목표**: 세 학습방법의 차이를 표로 비교·암기한다.
- 비지도: 데이터 내재 구조 자동 구분(군집·차원축소).
- 강화: 시행착오+보상으로 최적 행동 전략 학습.
- **3대 학습 비교(핵심 정리)**: 데이터/학습목표/활용범위/대표알고리즘이 모두 다름. 실제는 결합 사용도 가능.
**강의포인트**: 표를 빈칸 채우기로 반복 → 1차 필기 빈출 구간.
**예상문제(O/X)**: 강화학습은 정답 라벨이 주어진 데이터로 학습한다. → **X** (보상 기반).
