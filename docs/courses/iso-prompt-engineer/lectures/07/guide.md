## 도입 (멘트)
> "현실의 데이터는 대부분 정답이 없습니다. 그래서 비지도(패턴 찾기)와 강화(보상으로 배우기)가 중요하죠. 오늘은 세 학습을 한 표로 끝냅니다."

## 전개 (슬라이드별 강의 포인트)
### S4~5 비지도·강화
- 비지도: 군집·차원축소(구조 발견). 강화: 시행착오+보상.
### S6 비교표
- 지도/비지도/강화를 데이터·목표·피드백으로 비교(빈칸 채우기).
### S7 평가 차이
- 지도=정확도·정밀도·재현율·MSE(객관) vs 비지도=실루엣·WCSS(간접). 빈출.
### S8~9 비용·범위
- 라벨 비용(지도↑), 계산 비용(비지도 가능성↑), 활용 범위(예측 vs 탐색), 결합 사용.

## 활동 / 토론 (3분)
- "정답 없는 우리 데이터로 무엇을 할 수 있나?"(군집/이상탐지 아이디어).

## 정리 (멘트)
> "지도는 정답·오차, 비지도는 패턴·간접지표, 강화는 보상·정책. 다음 강부터는 실제 응용 — 자연어 처리(NLP)로 들어갑니다."

## 예상문제 + 해설
1. **(4지선다)** 군집 품질을 보는 비지도 지표는? → **실루엣 계수**(또는 WCSS).
2. **(O/X)** 비지도학습은 라벨 비용이 들지 않는다. → **O**.
3. **(단답)** 지도학습의 대표 분류 지표 2개는? → **정밀도·재현율**.
> 해설 포인트: 평가지표가 학습 종류별로 다름(객관 vs 간접).
