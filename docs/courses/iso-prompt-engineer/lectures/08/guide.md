## 도입 (멘트)
> "지금까지가 'AI가 어떻게 배우나'였다면, 이제부터는 'AI가 무엇을 하나' — 실제 응용입니다. 첫 번째는 우리와 가장 가까운 자연어 처리, NLP입니다."

## 전개 (슬라이드별 강의 포인트)
### S4 정의
- NLP = 언어를 이해·분석·생성. 인간-기계 소통 혁신.
### S5 번역
- 규칙기반 → **NMT**(문맥·의미). 중의성 예시로 문맥 이해 강조.
### S6 챗봇
- 규칙 → 딥러닝 생성형(의도 파악·자연 응답).
### S7~8 요약
- **추출적(발췌) vs 생성적(새 문장)** 표로 구분. "원문 그대로=추출적" 빈출.

## 활동 / 토론 (3분)
- "내 업무에서 자동 요약하면 좋은 문서?"(회의록·메일·리뷰 등).

## 정리 (멘트)
> "번역은 NMT로 문맥까지, 챗봇은 생성형으로 자연스럽게, 요약은 추출적/생성적 두 방식. 다음 강은 NLP 나머지 응용(감정·NER·QA 등)을 정리합니다."

## 예상문제 + 해설
1. **(4지선다)** 원문 문장을 그대로 뽑아 요약하는 방식은? → **추출적**.
2. **(O/X)** NMT는 문장 전체 문맥을 반영한다. → **O**.
3. **(단답)** 규칙 기반 번역을 대체한 신경망 기반 번역의 약어는? → **NMT**.
> 해설 포인트: 추출 vs 생성, NMT의 문맥 이해.
