## 전개 (슬라이드별 강의 포인트)
### S4 기준
- 정확성·완결성·형식준수·톤.
### S5~6 측정·워크플로우
- 체크리스트 채점, A/B, 일관성 / 작성→평가→수정→재실행.
### S7~8 개선·A/B
- 구체화·형식명시·요구나열, A/B 표로 선택.

## 실습 (6분)
