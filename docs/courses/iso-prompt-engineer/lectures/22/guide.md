# 강 22 — 1차 모의고사 + 해설 · 강의 스크립트

> AI Prompt Engineer · ISO/IEC 17024 · 1강 패키지 (강사용 운영안) · 교재 전체

## 강의 개요
- **시간**: 약 80분 (풀이 50분 + 해설 30분)
- **목표**: 실전 40문항 풀이 + 과목별 약점 진단
- **선수**: 강2~21 / **산출물**: 채점 결과·약점 분석표
- **자료**: `questions.md`(40문항·정답·해설)

## 운영 절차
1. **배포**: questions.md(또는 인쇄본) 배포.
2. **풀이(50분)**: 개인별 실전 풀이, 정답 보지 않기.
3. **채점(10분)**: "정답 요약"으로 자가 채점(2.5점×40).
4. **해설(30분)**: 오답률 높은 문항 위주 강사 해설.
5. **약점 분석**: 과목 태그별 오답 수 집계 → 복습 매핑.

## 합격 기준
- **24문항(60점) 이상** 합격.
- 과목 태그: 이해/이론/활용/윤리/활용실제/도구.

## 자주 틀리는 함정 (해설 강조점)
- CNN(이미지) ↔ RNN(순차) 혼동.
- AR(현실+디지털) ↔ VR(완전 가상).
- 지도(라벨) ↔ 비지도(무라벨).
- RLHF 3단계 순서(SFT→보상모델→강화학습).
- 딥페이크=GAN, 양면성.
