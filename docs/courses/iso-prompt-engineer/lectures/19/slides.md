## 핵심 개념 (1)
- **딥페이크(Deepfake)**: Deep Learning + Fake. **GAN(생성적 적대 신경망)** 기반으로 얼굴·음성을 조작·합성.
- **GAN 작동**: **생성자(Generator)**(위조지폐범) vs **판별자(Discriminator)**(경찰)가 경쟁 학습 → 점점 정교한 가짜.

## 핵심 개념 (2)
