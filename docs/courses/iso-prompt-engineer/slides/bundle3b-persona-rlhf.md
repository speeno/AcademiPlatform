# 강 16 · 페르소나 기반 프롬프팅

---

## 페르소나 프롬프팅
- **정의**: AI에 성격·직업·전문성·관점을 **명시 부여**(='가면')
- **목적**: 답변의 **스타일·전문성·눈높이** 제어로 '질' 향상
- **활용**: "너는 이제부터 ~ 전문가야 / ~의 관점에서 답해 줘"

---

## 페르소나 예시
- 전문가: "스타트업 경영 컨설턴트 → 시장 리스크·해결책"
- 교육자: "초등 5학년 선생님 → 지구 온난화 쉽게"
- 인물: "소크라테스 → '행복이란?'"
- 마케터: "MZ 디지털 마케터 → 숏폼 전략"

# 강 18 · RLHF

---

## RLHF — 인간 피드백 강화학습
- 강화학습 + **인간 피드백** → '예측기'를 '인간 가치 비서'로
- **3단계**:
  1. **SFT**(지도 미세조정): 형식·스타일 학습
  2. **보상 모델**: 인간 선호를 점수화 (핵심·'가치관')
  3. **강화학습(PPO)**: 보상 최대화로 최종 튜닝

---

## RLHF의 결정적 역할
- 🛡️ 안전성·윤리(유해 답변 억제)
- 🎯 미묘한 맥락 파악
- ✅ **환각 제어**(사실 기반 유도)
- 🎨 창의·유용성 품질 향상
