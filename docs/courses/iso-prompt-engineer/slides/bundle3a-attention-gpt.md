# 강 13 · 어텐션 ① 등장 배경

---

## 어텐션이 필요한 이유
- **어텐션**: 관련 깊은 부분에 **집중(가중치)** — "어디에 집중할까"
- **RNN/LSTM**: 순차 처리 + 히든 스테이트에 맥락 압축
- **한계**: 긴 문장에서 **병목·장기 의존성** → 앞 정보 손실

> 예: "그것은 맛있었다"의 '그것' = 햄버거

<!-- _class: section -->
# 강 14 · 어텐션 ② 셀프어텐션

---

## 셀프어텐션 — Q·K·V (도서관 비유)
- **쿼리(Query)**: 내가 찾는 것? (질문)
- **키(Key)**: 나는 이런 정보를 가졌다 (라벨)
- **값(Value)**: 실제 내용물
- 쿼리·키 비교 → **어텐션 스코어** → 값 가중 평균

---

## 어텐션이 가져온 혁신
1. **병렬 처리**(GPU) → 학습 속도 ↑
2. **장기 의존성 해결** → 긴 문맥 보존
3. **LLM 탄생** → GPT·BERT의 기반

> 셀프어텐션 = **트랜스포머의 심장**

# 강 15 · GPT vs 전통 검색

---

## 패러다임 차이
- 전통 검색: **키워드·링크로 문서 연결**
- GPT: **문맥 이해로 새 정보 창조**

| | 전통 검색 | GPT |
|---|---|---|
| 출처 | 명확 ✅ | 불명확 |
| 최신성 | 실시간 ✅ | 학습시점 한계 |
| 답변 | 링크 나열 | 정제된 답 ✅ |
| 창의성 | 약함 | 강함 ✅ |

---

## 장단점 요약
- **전통 검색 장점**: 출처·다양성·최신성·통제권
- **전통 검색 한계**: 비효율·문맥불일치·과부하·창의성부족
- **GPT 장점**: 즉각 맞춤답변·논리설명·창작·멀티모달(4o)
- **GPT 한계**: 출처 불명·환각 → **검증 필수**
