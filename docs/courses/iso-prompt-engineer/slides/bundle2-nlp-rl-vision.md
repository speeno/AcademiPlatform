# 강 9 · 자연어처리(NLP) ②

---

## NLP 주요 응용
- **감정분석**: 긍/부정·세부감정 / **정보추출(IE)**: 구조화
- **NER**: 사람·장소·날짜·기관 식별
- **QA**: 정확한 답 제시 / **토픽 모델링**: 숨은 주제 발견
- **맞춤법·문법 교정** / **STT+NLG**: 음성↔텍스트(시리·빅스비)

> 빈출: "서울시청=장소" 식별 → **NER**
